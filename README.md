## â“ Troubleshooting

**Links Not Working:**
- Check file names are lowercase
- Ensure all HTML files are in same directory
- Use relative paths: `projects.html` (not `/projects.html`)

**Images Not Showing:**
- Use full URLs for external images
- Or add images/ folder with image files

**Form Not Working:**
- Replace form action with valid endpoint
- Test with Formspree.io (free form backend)

**AdSense Not Showing Ads:**
- Ensure AdSense code is properly inserted
- Wait 24-48 hours after approval
- Check ad placement doesn't violate policies
- Verify domain is in AdSense account
